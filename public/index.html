Installation Script

This is your one-line installation script for setting up a Node.js application on the Linux server you want to manage. Please ensure you run this script with sudo permissions. Note that it is important to keep the script secure, as it includes a token. Additionally, be aware that this setup is insecure because it uses HTTP instead of HTTPS, and tokens ARE stored in cleartext. Use it with caution.
